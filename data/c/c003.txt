windows上sublime怎么运行c语言？
----------------------------------------
已经安装过gcc并添加到PATH。是perl语言自带的gcc，版本号是 gcc (x86_64-posix-seh, Built by strawberryperl.com project) 7.1.0
Copyright (C) 2017 Free Software Foundation, Inc.

1.菜单 tools-build system，选择new build system，在打开的配置文件输入如下内容，保存为myC.sublime-build。

{
    //"shell_cmd": "make"
    "working_dir": "$file_path",
    "cmd": "gcc -Wall \"$file_name\" -o \"$file_base_name\"",
    "file_regex": "^(..[^:]*):([0-9]+):?([0-9]+)?:? (.*)$",
    "selector": "source.c,source.cpp",

    "variants": 
    [
        {   
        "name": "Run",
            "shell_cmd": "gcc -Wall \"$file\" -o \"$file_base_name\" && \"${file_path}/${file_base_name}\""
        }
    ]
}

2.菜单 tools-build system，选择第一个自动。
3.ctrl+B执行文件，第一次可能需要选择命令，选择 gcc run

hello.c 文件

#include<stdio.h>
int main(){
	printf("hello, C\n");
	return 0;
}

即可看到运行结果
hello, C
[Finished in 0.4s]
