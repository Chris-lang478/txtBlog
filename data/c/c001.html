步骤：
1.先安装vim
$sudo apt-get install  vim
2.编辑自己的hello word
$vim  hello.c
3.输入小写英文 "i" 就可以进行编辑，输入如下内容
#include <stdio.h>
int main(){
     printf("hello,word");
     return  0;
}
4.编写完成后 按esc键 然后输入 ":wq!" 然后enter进行保存

5.下面是进行编译,编译后会多出一个编译好的可执行程序文件
gcc编译C语言的最简命令为：gcc c_files -o target_name

$sudo gcc -o  编译后的名字 hello.c
