安装与使用

参考：http://knpuniversity.com/screencast/composer

win7环境。使用的git的命令行界面，类linux命令，很好用！。

1.下载安装器
win7用浏览器：https://getcomposer.org/installer


2.移动下载的文件到web中某目录下。配置php的环境变量（请百度）。
然后执行
	$ ls
	installer

	$ php installer
	All settings correct for using Composer
	Downloading...

	Composer successfully installed to: F:\xampp\htdocs\composer\composer.phar
	Use it: php composer.phar
