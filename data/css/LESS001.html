LESS描述

Less (LESS官网[2.5.3] An overview of Less, how to download and use, examples and more.) 是一门 CSS 预处理语言，它扩充了 CSS 语言，增加了诸如变量、混合（mixin）、函数等功能，让 CSS 更易维护、方便制作主题、扩充。

Less 可以运行在 Node、浏览器和 Rhino 平台上。网上有很多第三方工具帮助你编译 Less 源码。

http://www.bootcss.com/p/lesscss/ [1.3.1]一种 动态 样式 语言.

http://www.1024i.com/demo/less/ LESSCSS是一种动态样式语言，属于CSS预处理语言的一种，它使用类似CSS的语法，为CSS的赋予了动态语言的特性，如变量、继承、运算、函数等，更方便CSS的编写和维护。

http://less.bootcss.com/  快速入门

http://www.ibm.com/developerworks/cn/web/1207_zhaoch_lesscss/ LESS CSS 框架简介, 使用 LESS 简化层叠样式表（CSS）的编写

http://www.w3cplus.com/css/less 教程。
http://www.lesscss.net/ 如何在服务器端、客户端和第三方环境中使用 Less.

客户端使用LESS

在浏览器上跑 less.js 非常方便开发，但是不推荐用于生产环境。在生产环境中，性能和可靠性非常重要， 我们建议最好使用 node.js 或其它第三方工具进行预编译。

<script src="//cdnjs.cloudflare.com/ajax/libs/less.js/2.5.3/less.min.js"></script>
