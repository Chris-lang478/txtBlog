博客示例

该博客系统设计目标是要支持txt格式的文件，方便组织和阅读我的笔记。未来，可能会支持MarkDown等更多格式。

目前支持html格式笔记，具有简单的html格式化显示功能。本文是示例。

这是强调文字。这是说明文字。

/*
	用于调试，打印php中的变量
*/
function debug($para,$isDie=true){
	//echo '这是代码区域';
	echo '<pre>';
	print_r($para);
	echo '</pre>';
	if($isDie){
		die();
	}
}
