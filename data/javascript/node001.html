项目描述

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。 Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。Node.js 的包管理器 npm，是全球最大的开源库生态系统。
