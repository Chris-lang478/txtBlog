sublime简介

Sublime Text is a sophisticated text editor for code, markup and prose. 特点：常见代码高亮、自动完成、众多插件、自定义配置等。

You'll love the slick user interface, extraordinary features and amazing performance. 更多信息查看非官方教程。

ubuntu1404安装sublime

http://www.csdn123.com/html/blogs/20130923/73206.htm
一、指令安装
	sudo add-apt-repository ppa:webupd8team/sublime-text-2
	sudo apt-get update
	sudo apt-get install sublime-text-2
二、安装包
  1. 下载：$wget https://download.sublimetext.com/Sublime%20Text%202.0.2%20x64.tar.bz2
  2. 解压：$tar jxvf Sublime\ Text\ 2.0.2\ x64.tar.bz2
    解压后到安装包重新命名为sublime2:$ mv Sublime\ Text\ 2 Sublime2
  3. 将sublime移至安装目录:/usr/lib: $ sudo mv Sublime2 /usr/lib/
  4. 创建快捷方式:/usr/lib: $sudo ln -s /usr/lib/Sublime2/sublime_text /usr/bin/sublime
    这样可以在命令行中使用：$sublime打开编辑器。
  5. 创建启动器:$sudo sublime /usr/share/applications/sublime.desktop， 内容如下：
  6. 把启动器图标从/usr/share/applications/中复制到桌面，即可双击启动。
[Desktop Entry]
Version=1.0
Name=Sublime Text 2
# Only KDE 4 seems to use GenericName, so we reuse the KDE strings.
# From Ubuntu's language-pack-kde-XX-base packages, version 13.04-20130413.
GenericName=Text Editor

Exec=sublime
Terminal=false
Icon=/usr/lib/Sublime2/Icon/48x48/sublime_text.png
Type=Application
Categories=TextEditor;IDE;Development
X-Ayatana-Desktop-Shortcuts=NewWindow

[NewWindow Shortcut Group]
Name=New Window
Exec=sublime -n
TargetEnvironment=Unity

常用操作

http://blog.sina.com.cn/s/blog_7d34486c0100vu20.html

http://www.cr173.com/html/30661_1.html

整行上下移动：ctrl shift 上下

整行复制到并插入到下一行：ctrl shift D

删除整行：ctrl shift K, Ctrl+X剪切整行

整行左右移动：按 Ctrl + [ 整行向左移动；按 Ctrl + ] 整行向右移动

选取整列：shift+右键；ctrl增加选区。
Shift 鼠标右键 (Win) 或 Option 鼠标左键 (Mac) 或使用鼠标中键可以用鼠标进行竖向多行选择；

Ctrl 鼠标左键(Win) 或 Command 鼠标左键(Mac) 可以手动选择同时要编辑的多处文本

鼠标选中文本，反复按 CTRL D (Command D) 即可继续向下同时选中下一个相同的文本进行同时编辑；

鼠标选中多行，按下 Ctrl Shift L (Command Shift L) 即可同时编辑这些行；

鼠标选中文本，按下 Alt F3 (Win) 或 Ctrl Command G(Mac) 即可一次性选择全部的相同文本进行同时编辑；

ctrl+shifl+/ 注释所选内容

ctrl+/ 注释当前行

ctrl+G 跳到指定行

先按下快捷键 ctrl+` 调出命令行，输入：

import urllib2,os;pf='Package Control.sublime-package';ipp=sublime.installed_packages_path();os.makedirs(ipp) if not os.path.exists(ipp) else None;open(os.path.join(ipp,pf),'wb').write(urllib2.urlopen('http://sublime.wbond.net/'+pf.replace(' ','%20')).read())  

回车，安装结束后重启。在Perferences->package settings中看到package control这一项，则安装成功。
