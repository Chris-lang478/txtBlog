分辨率：
article PDFs are processed to display images at 200 ppi, which will guarantee quality at print size. 
The HTML display of an article offers 300 ppi for all images. 


(1) 三种图片大小
Small: approximately 9 cm x 6 cm
Medium: approximately 11 cm x 11 cm
Large: approximately 18 cm x 22 cm


(2) 文字大小 6pt-12pt.
Ensure that all numbers, letters, and symbols are no smaller than 6 points (2 mm) and no larger than 12 points (6 mm) after reduction. Keep text sizing consistent within each graphic.

预先整合图片，输出 TIFF, EPS, PDF 格式。
Preassemble all composite figures. 
Submit images in these file formats: TIFF, EPS, PDF, or PPT.

2. 细则
(1) 标签和文字的使用：嵌入字体文件。
Labeling and font usage: Please use the same font for all figures in your manuscript. 

使用如下字体：arial, ...
Use standard fonts such as Arial, Helvetica, Times, Symbol, Mathematical Pi, and European Pi. Embed all fonts used in vector files. 
In Illustrator, check the box that reads Embed Fonts when saving the file. See page 3 for more detailed instructions on embedding fonts.


(2) 尺寸、字体大小建议：
- 按最终大小准备图片，防止缩小不能看。
- 高度要考虑给图例留点地方：
Recommended max height: 54 picas / 9” / 22.5 cm.
Use one of the following widths:
1 column wide (20.5 picas / 3.42” / 8.7 cm)
1.5 columns wide (27 picas / 4.5” / 11.4 cm)
2 columns wide (42.125 picas / 7” / 17.8 cm)
* For example, if you are creating a combination figure (image+text) that you want to be 1 column wide using Adobe Illustrator (which is preferred to maintain vector text), choose File>New... and enter these settings (shown at right):
组合图片和文字，1列宽，

- 图片位置放好 Place artwork onto the canvas/background and position appropriately
- 字体大小 6-8点。 Size any text in your figure to at least 6–8 points. This will allow us to reduce the file if necessary.

- 修改高度：PS和AI。
If you need to crop the height of your figure, in Adobe Illustrator, choose File>Document Setup, then either Edit Artboards to crop the artboard to size, or change the size of the document as needed. 

If you are using Photoshop, crop the canvas down to the size needed using the crop tool.

- 缩小现有图的体积
If it is necessary to reduce the size of existing figures:
	* 去掉没用的白背景
	* PS，Image>Image Size 调整大小。Uncheck Resample Image，防止PS插入不能提升图片质量的像素。
	* AI, Object>Transform>Scale..，然后选择 Uniform Scale，然后输入需要的百分比，勾选 Scale Strokes and Effects。
	* 保证字体至少6 pints. If it is not, please use the text tool to reset the text before submission


- 矢量图
	* AI 矢量图最好保存为 EPS 文件，插入所有字体，线至少 0.25points 宽。如果使用AI，保存文件时，确保勾上 Embed Fonts。
	* Combination line/halftone: 栅格图用PS，然后在AI加上文字和连线，输出位 EPS 文件，插入所有字体，线至少 0.25poinits 宽。

- MS office: 不推荐，否则可能延期。如果非要用，注意几点：
	* 不要用 pattern or textured 文件。使用  solid fills or percentage screens，方便转为矢量图。注意：至少差异 20% 是比较理想的。
	* 300 dpi for halftones, 600–900 dpi for combinations, and 1000–1200 dpi for line art.
	* 插入图片到文件时，选择 “insert” 而不是 “insert link.”。后者不能正确插入高清图片到MS office。
	* 对于 office 创建的文件，保存前选择  File>Options>Advanced，勾选“Do not compress images in file.” 这会保持高清图片不变小。
	* 不要输出 office 文件为 tif 格式: 这会栅格化矢量文字和线条。
	* 在 office 中插入字体文件:
		- 菜单 file menu, select Save As...
		- 菜单 Tools menu, select Save Options，select the Embed Fonts check box，and select Embed all characters (best for editing by others).

- 3D Images 跳过。









(3) 图例 Figure legends
Include figure legends immediately after referencing the figure in the manuscript. Ensure that figure legends adhere to the following guidelines: 

- For figures with multiple panels, the first sentence of the legend should be a brief overview of the entire figure. Explicitly reference and describe each panel at least once in the figure legend.
- Include clearly labeled error bars in all graphs and describe them in the figure legend.
- State whether a number that follows the ± sign is a standard error (SEM) or a standard deviation (SD).
- When applicable, provide the P value, magnification, or scale bar information. 
- Indicate the number of independent data points (N) represented in a graph in the legend.
- Ensure that numerical axes on all graphs go to 0, except for log axes.


必须是 RGB 颜色。


位图：300 ppi for grayscale or color images with no type or lettering, 600–900 ppi for grayscale or color images with type, and 1000–1200 ppi for line art, e.g., bar graphs.
Avoid creating text in graphics.

单色的：1000 and 1200 ppi at publication size
半拼合： 600 and 900 ppi at publication size
没有文字的： 300 ppi at publication size. 

Note: When saving these files, please use LZW compression (i.e., do not use jpeg compression).
ps 保存为tif时，可以选择lzw压缩。

TIFF所有压缩方法都是无损压缩，压缩的效果很低。JPG是采用有损压缩，通过降低图像质量达到空间的压缩，所以对于真彩色图像，TIFF不管怎么压缩，容量都会远远大于JPG图像。

图片排版的技术流程: 经验与教训总结
----------------------------------------
散点图或者线图，如果很复杂，整个图就会很大。
总体思路是：可以先截图，再添加文字。


1. 图片生成后，先用AI打开看看文字是否变形。
放到1:1 A4纸，看文字大小是否能看清。


2. 尺寸要掌握好
页面尺寸: 18cm宽，高度 22cm以内。
图片尺寸: pdf("xx.pdf", width=3, height=3) #3-5之间，一般不能再大了



3. 缩小散点图尺寸的方法: 用 illustrator 排版
(1) pdf 图大小固定后，替换点图为 散点图: 4倍放大截屏，去掉黑box并缩小为原来的50%，粘贴回去再缩放50%。
这时尺寸是对的，只需要上下左右移动到合适位置。
删掉原来的散点图 图层。



3. 补充材料图片整理

(1) 先在 ppt 中排好位置和大小
ppt设置 17cm * 20cm。

(2) 全选一页，复制粘贴到word中
右击-编组
右击-环绕-嵌入。

$ du -sh pngimage.png
$ pngquant pngimage.png
$ du -sh pngimage-fs8.png
