然而，随着作图要求的提升，传统的二维图形编辑软件（如PS、AI等）已渐渐难以满足大众的需求。与此同时，三维制图软件在国内多用于建筑设计、影视动画、游戏角色建模等领域，在科研绘图领域的应用尚处于起步阶段。

3D建模类软件主要有3ds Max、Maya、Cinema 4D、Blender等，基本操作原理都是类似的，只是操作习惯有所不同。这些软件都具备系统的建模、材质、灯光和动画模块，但要做到精通任何一款软件，大量的练习和思考是必不可少的。3ds Max的优势在于教程资源丰富，兼容性强，而且拥有自身的MAXScript语言，提供了开放式操作的可能性。 (在这个“刷脸”的时代，发Paper也要拼“颜值” CCL 研之成理 2018.7月29日)

Blender（斑斓、搅拌机）

Blender(Blender中文论坛 / Blender Reference Manual / Blender Python API手册) 是一款开源的跨平台全能三维动画制作软件，提供从建模、动画、材质、渲染、到音频处理、视频剪辑等一系列动画短片制作解决方案。

Blender 拥有方便在不同工作下使用的多种用户界面，内置绿屏抠像、摄像机反向跟踪、遮罩处理、后期结点合成等高级影视解决方案。同时还内置有卡通描边（FreeStyle）和基于 GPU 技术 Cycles 渲染器。以 Python 为内建脚本，支持多种第三方渲染器。

Blender 为全世界的媒体工作者和艺术家而设计，可以被用来进行 3D 可视化，同时也可以创作广播和电影级品质的视频，另外内置的实时 3D 游戏引擎，让制作独立回放的 3D 互动内容成为可能。

有了 Blender 后，喜欢 3D 绘图的玩家们不用花大钱，也可以制作出自己喜爱的 3D 模型了。它不仅支持各种多边形建模，也能做出动画！

难点：

像Blender这样的3D内容创作工具软件在深层的技术背后还有着不少技术难点以及行业术语。一些专用术语比如UV贴图、材质、着色器、网格、细分曲面等都是数字艺术家表达思想的媒介，理解它们，运用它们，你才能将Blender用到极致。

操作上，blender这个3D软件，更倾向于尽可能使用快捷键而不是去找按钮点击。因此，你应该把“熟练快捷键”作为学习目标之一。
